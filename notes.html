Tags

HTML uses tags as it's basic method of formatting.
Tags represent things involving formatting such as < br > which represents
page
breaks.

There are alot of other tags. < img > is a tag that is used to direct the broswer to the source file of an image and place it in the browser window.

Also, you can create a link using the < a > tag.
You simply type < a href="yourlinkhere"> then the text you want as a link
< /a > then the closing bracket.

HTML Document Structure

HTML documents have two basic sections.

The first is the < head > head section. The head section is the section that all of your first run stuff goes, such as Javascript and CSS code. Another thing you want to place in your head section is the < title > tag. The title tag is where your browser looks to populate the little bar at the top of the window when you're visiting a webpage.

The second main section is the < body > body section.
The body section is where you want to place all of your text, images, links, etc. It is where the real meat of your page goes.
